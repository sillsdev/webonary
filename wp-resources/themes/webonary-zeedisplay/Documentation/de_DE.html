A) Anforderungen - top

Dieses Theme wurde auf allen gebräuchlichen Browser Firefox 2 & 3.6, Opera 9.5 & 10, Internet Explorer 7 & 8 und Safari 3.2 getestet. Um alle Funktionen des Themes vollständig zu nutzen, sollte JavaScript in deinem Browser aktiviert sein.

Das Theme funktioniert mit der aktuellen Wordpress Version ohne Probleme. Vorherige Versionen wurden nicht getestet und werden nicht unterstützt. Ich schlage auch aus Sicherheitsgründen vor, die Wordpress Version stets aktuell zu halten.

B) Installation - top

  1. Automatische Installation

    Gehe zu WP-Admin > Design > Themes > Themes installieren > Hochladen und wähle den mitgelieferten Theme Zip Folder aus.

  2. Manuelle Installation

    Lade den Theme Ordner und alle Dateien via FTP in den Wordpress Theme Ordner wp-content/themes hoch und aktiviere das Theme im Wordpress Admin Panel. Fertig!

Mehr Informationen zur Installation von Wordpress Themes findest du in der WordPress Dokumentation.

C) Wordpress Optionen - top

Anschließend ein paar weitere Erklärungen und empfohlene Einstellungen für das Theme

  1. Post Thumbnails

    Das Theme verwendet die Wordpress post_thumbnail Funktion. Damit kann jedem Artikel ein Thumbnail Bild hinzugefügt werden. Beim Erstellen eines Artikels befindet sich auf der rechten Seite in der Artikelbild Box die Funktion Artikelbild festlegen. Dort kann ein Bild hochgeladen werden und mit der Funktion Als Artikelbild nutzen eingefügt werden.

  2. Bildeinstellungen

    Die Größe der Thumbnails wird von den Wordpress Mediathek Einstellungen bestimmt. Dies können unter WP-Admin > Einstellungen > Mediathek geändert werden. Das Theme verwendet die Miniaturbildgröße in der Artikel- und Seitenansicht und die Mittlere Bildgröße in den Übersichts- und Archivseiten.

  3. Seitenzahlen

    Das Theme beinhaltet CSS Styles für das 'wp_pagenavi' Plugin. Um die Seitenzahlen anzuzeigen kannst du dir das Plugin hier herunterladen und installieren.

D) Theme Optionen - top

Das Theme hat ein eigenes Theme Admin Panel für weitere Einstellungen. Dieses ist unter WP-Admin > Design > “zeeDisplay” Optionen zu finden. Dort findest du zahlreiche Einstellungen zur Änderung des Logos, der Struktur sowie Layout des Themes. Des Weiteren kann hier der Featured Post Slider und die 125x125 Banner verwaltet werden. Jede Option ist ausführlich beschrieben, sodass hier keine Fragen auftauchen sollten.

Die Einstellungen sollten die Einrichtungszeit deines Wordpress Blogs deutlich reduzieren. Für tiefergreifende Änderungen müssen die Dateien manuell geändert werden.

E) Theme Farben - top

Das Theme beinhaltet bereits einige vordefinierte Farbkombinationen. Ebenfalls kann mit der 'custom color' Funktion ein eigener Farbcode im Theme Admin Panel eingetragen werden.

Alle Styles sind in dem Ordner /includes/styles gespeichert. Dort besteht für jede Farbe eine CSS Datei und ein Bildordner, die den allgemeinen CSS Code überschreiben. Hier kannst du sehr leicht manuell deinen ausgewählten Style ändern oder sogar neue hinzufügen. Damit besteht die Möglichkeit, das Theme noch individueller auf deine Wünsche anzupassen.

F) Page Templates - top

zeeDisplay unterstützt ein paar alternative Page Templates

  1. Full Width Template

    Zeigt eine Seite in voller Breite ohne Sidebars.

  2. Sitemap Template

    Zeigt eine Sitemap, wo alle Artikel, Seiten und Kategorien aufgelistet werden.
